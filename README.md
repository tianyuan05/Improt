# Improt

## 基本功能说明

#### 1.登陆功能，若账号拥有多个店铺，则可以进行切换当前登陆店铺

#### 2.店铺会员查询
        2-1.查询单个会员的详细信息（会员基本信息+会员在店铺购买商品信息）；
        2-2.查询当前店铺会员列表；
        
#### 3.从格式化的Excel中读取数据
         3-1.是否具有表头；
         3-2.读取的错误数据输出日志，读取不中断；
         3-3.读取有错误数据不尽兴第4步；
     
#### 4.导入读取数据
          4-1.保存失败的数据输出日志，保存不中断；
          4-2.统计导入成功的记录数；
